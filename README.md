# 智慧排班系統 (Web-based Smart Scheduler) V2.3

這是一個基於 Python Flask 和 Google OR-Tools 的智慧醫師排班網頁應用程式。系統旨在將一個複雜的、基於多重規則與優化目標的排班問題，透過現代化的網頁介面進行管理，實現從醫師提交預休到排班人員一鍵生成最佳化班表的無縫流程。

## ✨ 核心功能 (V2)

* **動態醫師名單**:
    * **醫師端**: 不再需要從固定選單選擇，醫師可直接**輸入姓名**登入個人化儀表板。
    * **管理員端**: 擁有完整的「總醫師設定面板」，可以**動態新增/刪除**當月醫師，並為每位醫師指派**區域**與**點數上限**。

* **自動化跨月規則**:
    * 系統會自動偵測醫師**上個月最後兩天的排班狀況**，並在當月自動執行「班後需連休兩天」的硬性規則，無需總醫師手動介入。

* **優化的使用者體驗**:
    * **醫師端**:
        * 即時計算並顯示目前已選取的**預休點數**，超過 4 點會出現提醒。
        * 已提交的預休會處於**唯讀狀態**，需點擊「修改」按鈕才能重新編輯，防止誤觸。
    * **管理員端**:
        * 醫師設定列表會以**顏色區分不同區域**的醫師。
        * 列表會自動排序，優先按**區域**分組，同區域內再按**點數上限**由高至低排列。
        * 支援**拖曳醫師**來快速變更其所屬區域。
    * **開發者除錯面板**: 提供「一鍵載入預設範本」和「一鍵清空當月資料」的測試功能。

* **一鍵智慧排班**:
    * 整合 Google OR-Tools 的 CP-SAT 求解器，追求最佳解。

* **即時運算日誌與視覺化結果**:
    * 即時串流後端運算過程，並呈現精美的視覺化班表。
    * 支援區域/日期篩選、全螢幕檢視，並可下載 Excel 報表。

* **資料持久化**:
    * 所有預休和設定資料都會儲存於伺服器的 `data.json` 檔案中。

## 🛠️ 技術架構

* **後端**: Flask, Google OR-Tools, Pandas, openpyxl, holidays, gevent
* **前端**: HTML, CSS, JavaScript (Vanilla JS), Bootstrap 5.3, SortableJS
* **即時通訊**: Server-Sent Events (SSE)

## 📖 使用指南 (V2)

### 醫師操作流程

1.  在首頁選擇「我是醫師」。
2.  在左側欄位**輸入您的姓名**，並點擊「確認」。
3.  右側會出現當月份的日曆。
    * **如果本月尚未提交預休**: 日曆處於**可編輯狀態**。您可以點擊或拖曳來選取預休日期。左側會即時顯示預休點數。完成後點擊右下角的「提交本月預休」。
    * **如果本月已提交預休**: 日曆會處於**唯讀狀態**，顯示您已提交的日期。如需修改，請點擊右下角的「修改預休」按鈕，日曆會解鎖變為可編輯狀態。
4.  修改完成後，再次點擊「提交本月預休」即可儲存。

### 排班管理員操作流程

1.  在首頁選擇「我是排班管理員」。
2.  **設定醫師**:
    * 畫面會顯示當月已提交預休的醫師列表。
    * 您可以手動**新增醫師**，或**刪除**某位醫師。
    * 為每位醫師選擇**區域 (A/B/C/I)** 和設定**點數上限**。
    * 列表會自動按區域（帶有顏色）和點數排序。您可以直接**拖曳**醫師到不同區域的群組來快速修改其區域。
3.  **儲存設定**: 完成所有醫師的設定後，點擊左側的「**儲存醫師設定**」。此按鈕非常重要，必須點擊後才能啟用排班功能。
4.  **一鍵排班**: 儲存後，「一鍵排班」按鈕會變為可點擊狀態。點擊後，下方的「即時運算日誌」區塊會開始顯示求解過程。
5.  **檢視結果**: 運算完成後，結果會顯示在下方，您可以切換頁籤檢視不同圖表，或下載完整的 Excel 報表。

### 開發者測試功能

* 在管理員頁面頂部，有一個可收合的「**開發者除錯面板**」。
* **載入預設醫師範本**: 點擊會將預設的 20 位醫師（包含區域、點數、預休日）載入到前端表格中。**此操作不會自動儲存**，您仍需手動點擊「儲存醫師設定」才會生效。
* **清空本月所有資料**: 點擊會**直接刪除**伺服器上儲存的當月所有資料，讓您可以從乾淨的狀態開始測試。
